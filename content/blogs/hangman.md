### Overview

[Hangman](https://en.wikipedia.org/wiki/Hangman_(game)) is a classic word-guessing game where the player tries to reveal a hidden word by guessing one letter at a time. Each correct guess uncovers part of the word, while each wrong guess brings the player closer to losing the game.

### Context

I made a small web-based version of Hangman using HTML, CSS and JavaScript. I wanted to recreate the core features of the traditional game in a browser-based format, while keeping the interface clear, playful and easy to use.

For the game logic, I used JavaScript to handle the main interactions. I created a fruit-themed word library for the game to draw from, so the hidden word is always a type of fruit. I also recreated the visual hangman progression by using a sequence of images that change as the player makes incorrect guesses.

To make the game more fun and accessible to play, I added support for both mouse clicks and keyboard input. The game can be played on both desktop screens and smaller screens. I also added sound effects for correct guesses, wrong guesses, winning and losing, so the game feels more responsive and satisfying as players move through each round.
